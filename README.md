# 📚 Library Book Manager (Python Console Project)

A beginner-friendly **Python console application** to manage library books using **functions, dictionaries, loops, conditionals, and file handling**.

This project was built as practice after learning Python basics.

---

## 🚀 Features

- ➕ Add new books
- 📖 View all books
- 📥 Borrow books
- 📤 Return books
- 🔍 Search books by name
- 💾 Save library data in a text file
- 🔁 Menu-driven program

---

## 🛠️ Technologies Used

- Python 3
- File Handling (`read`, `write`)
- Dictionaries
- Functions
- Loops (`while`)
- Conditional Statements (`if-elif-else`)
